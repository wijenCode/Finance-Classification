Indonesian Personal Finance Dataset dari Kaggle

Overview Dataset

Dataset yang digunakan dalam sistem ini adalah Indonesian Personal Finance Dataset yang tersedia di platform Kaggle. Dataset ini berisi data keuangan personal masyarakat Indonesia yang telah dilabeli untuk keperluan klasifikasi kesehatan finansial.

Lihat Dataset di Kaggle

Fitur-Fitur Dataset

Pemasukan (5 fitur)

  • Gaji: Pendapatan utama bulanan
  • Tabungan Lama: Tabungan yang sudah ada sebelumnya
  • Investasi: Pemasukan dari investasi
  • Pemasukan Lainnya: Sumber pendapatan tambahan
  • Tipe: Pola pengeluaran (hemat/normal/boros)

Pengeluaran Esensial (4 fitur)

  • Bahan Pokok: Kebutuhan makanan pokok
  • Protein & Gizi Tambahan: Makanan bergizi
  • Tempat Tinggal: Biaya sewa/cicilan rumah
  • Sandang: Kebutuhan pakaian

Pengeluaran Non-Esensial (3 fitur)

  • Konsumsi Praktis: Makanan siap saji
  • Barang & Jasa Sekunder: Kebutuhan sekunder
  • Pengeluaran Tidak Esensial: Hiburan, hobi

Kewajiban & Tabungan (4 fitur)

  • Pajak: Pembayaran pajak
  • Asuransi: Premi asuransi
  • Sosial & Budaya: Kegiatan sosial
  • Tabungan/Investasi: Alokasi tabungan baru

Total: 16 Fitur Input

Model Machine Learning

Sistem ini menggunakan 3 jenis model machine learning untuk memberikan fleksibilitas dalam analisis:

Boosting (XGBoost)

Teknik ensemble yang membangun model secara sekuensial, di mana setiap model baru mencoba memperbaiki kesalahan model sebelumnya. Sangat efektif untuk dataset kompleks dan memberikan akurasi tinggi.

Akurasi Tinggi Handling Overfitting

Bagging (Random Forest)

Teknik ensemble yang membangun beberapa model secara paralel dengan subset data yang berbeda, kemudian menggabungkan hasilnya. Random Forest sangat robust terhadap noise dan outliers.

Robust Mengurangi Variance

Deep Learning (Neural Network)

Model neural network yang mampu mempelajari pola kompleks dan non-linear dalam data. Sangat powerful untuk menangkap hubungan tersembunyi antar fitur finansial.

Pattern Recognition Non-linear Relations
